# 森之启示 - 3D 手势交互塔罗占卜系统 (Zhanbu)

这是一个基于现代 Web 技术构建的沉浸式 3D 塔罗牌占卜应用。结合了顶级的视觉设计（“奇幻森林”主题的毛玻璃风格）、强大的物理交互引擎（Three.js）以及前沿的 AI 计算机视觉技术（MediaPipe 手势追踪），为用户提供极其专业且极具仪式感的命运探索体验。

## ✨ 核心特性

- 🔮 **全息 3D 牌桌交互**：通过 React Three Fiber 渲染的 3D 环形星轨牌堆。每张卡牌在被抽取时都会展现极其丝滑、带有物理惯性的飞行动画。
- ✋ **魔法般的“手势追踪” (Hand Tracking)**：
  - 弃用传统的鼠标点击，接入 Google MediaPipe 视觉模型。
  - **捏取抽牌 (Pinch)**：通过摄像头捕捉食指与拇指的捏合，实现“隔空取牌”。
  - **握拳制动 (Fist Stop)**：当你握紧拳头时，命运的轮盘（3D 星轨）会立即停止旋转，增强占卜的仪式感。
- 📚 **学术级全量知识库**：
  - 内置完整的 78 张塔罗牌（22 张大阿卡纳 + 56 张小阿卡纳）高清图库（1536x2754 完美比例）。
  - **大师级专业解析**：每张牌均包含极具深度的心理学含义、行动建议及 RWS 经典符号的万字级拆解。
- 🔄 **正逆位命运系统**：完全模拟真实洗牌，30% 的概率出现卡牌“上下颠倒”（逆位）。UI 也会通过粉色边框和专属的逆位判词进行自适应反馈。
- 👁️ **渐进式奥义揭晓**：
  - **快卜预览**：在抽牌结束后，首先展示 50 字以内的精辟“核心概括”，让你一眼看清能量全貌。
  - **深度阅读**：点击“阅读奥义”，弹出精美的全屏毛玻璃弹窗，沉浸式阅览上千字的专业塔罗解析。

## 🛠️ 技术栈 (Tech Stack)

- **前端框架**: React 18 + TypeScript + Vite
- **3D 引擎**: Three.js + `@react-three/fiber` + `@react-three/drei`
- **动画系统**: GSAP (GreenSock)
- **视觉追踪**: `@mediapipe/tasks-vision` (手部关键点识别)
- **样式方案**: Tailwind CSS (自定义了大量主题变量与毛玻璃特效)
- **图标库**: Lucide React

## 📦 环境要求 (Requirements)

- **Node.js**: 建议版本 v18.0.0 或更高。
- **包管理器**: `npm` (或 `yarn`, `pnpm`)。
- **硬件要求**: 必须配备 **Web 摄像头 (Webcam)** 以启用手势交互功能。

## 🚀 启动方式 (Getting Started)

1. **克隆或下载项目**到本地。
2. **安装依赖**:
   在项目根目录下打开终端，运行：
   ```bash
   npm install
   ```
3. **启动开发服务器**:
   ```bash
   npm run dev
   ```
4. **体验项目**:
   - 打开终端提示的本地链接（通常为 `http://localhost:5173`）。
   - 浏览器会请求**摄像头权限**，请务必点击“允许”。
   - 等待几秒钟，直到左上角的指示灯变为翠绿色的 `LINK_ACTIVE`，即可开始使用手势进行占卜！

## 📖 如何使用 (How to Play)

1. **设定意图**：在首页选择你想要的牌阵（如单张、三张或凯尔特十字），并在心中默念你的问题。
2. **开始洗牌**：点击“注入意念，开始洗牌”，系统会自动为你打乱 78 张牌的顺序。
3. **隔空抽牌**：
   - 面对摄像头，确保你的手部清晰可见。
   - 看着屏幕中转动的星轨，当你感觉时机已到，**握紧拳头**让星轨停下。
   - 伸出食指和拇指，对准某张牌做出**“捏合”**动作，卡牌即会被你抽出并飞入前方的托盘中。
   - 当牌托盘填满（根据你选择的牌阵数量），系统将自动为你揭晓牌面。
4. **解读命运**：阅读右侧的精要总结，如果你需要更深层次的指导，点击任意一张卡牌下方的“阅读奥义”以获取专家级解析。
